This meant to be a somewhat up to date example of using Node JS to write an app that pulls the public Twitter stream. However, this will not be updated and I make no promises in regards to its quality for production use. 

Also, if you want to try it out yourself you will probably want to do so from a webpage of your own design (one is no included here).

Node JS is evolving quickly and this code may not work in a future versions of Node; it written for v0.5.0-pre
